Getting Started

Learn how simple it is to start using Paradox on your site

Download

Paradox is an open-source framework based on Material Design, To start Paradox, all you need to do is use one of the forms below

Paradox

This is the standard version that comes with both the minified and unminified CSS and JavaScript files. This option requires little to no setup. Use this if you are unfamiliar with Sass.
file_download Paradox Framework

Sass

The SCSS code is available. By choosing this version you have greater control over the included components. You need to use a Sass compiler if you choose this option
file_download Source

Import's

Paradox on your project

CDN

You can use UNPKG for files import 

            <!-- Compiled and minified CSS -->
            <link rel="stylesheet" href="https://unpkg.com/paradox@1.0.1/dist/css/paradox.min.css">

            <!-- Compiled and minified JavaScript -->
            <script src="https://unpkg.com/paradox@1.0.1/dist/js/paradox.min.js"></script>

Npm

You can also get the latest release using NPM 

            npm install paradox
